# 🌙 Atomic Habit & Task Tracker

En gratis, personlig habit tracker og task tracker bygget med React og Firebase — inspireret af bogen **Atomic Habits** af James Clear.

**Ingen abonnement. Ingen reklamer. Du ejer din egen data.**

---

## Hvad får du?

- ☀️ **Habit Tracker** — daglige vaner organiseret i pillarer (Deen, Arbejde, Krop, Fundament)
- ◑ **Ugentlig Plan** — sæt vaner til specifikke ugedage (træning kun man/ons/fre osv.)
- ◈ **Task Tracker** — opgaver med deadlines, prioriteter og kategorier
- ◻ **Kalender** — overblik over alle deadlines, eksport til iPhone Kalender
- 🔒 **Privat** — kun du har adgang via Google Login
- 🌗 **Lys/mørk tema** — følger automatisk din telefons indstilling
- 📱 **Virker på telefon og computer** — deploy én gang, brug overalt

---

## Sådan bruger du den

Du behøver ikke kunne kode. Du skal bare have:
- En Google konto
- Adgang til [Claude.ai](https://claude.ai)

### Trin 1 — Kopiér koden
Download eller kopiér filen `App.jsx` fra dette repo.

### Trin 2 — Åbn Claude
Gå til [claude.ai](https://claude.ai) og start en ny samtale.

### Trin 3 — Send denne besked til Claude

```
Jeg vil gerne have en personlig habit tracker og task tracker.
Jeg har vedhæftet koden til en færdig app som udgangspunkt.

Hjælp mig med at:
1. Tilpasse vanerne og kategorierne til mit eget liv
2. Sætte den op med Firebase (gratis hosting + login)
3. Deploye den så jeg kan bruge den på min telefon

[Vedhæft App.jsx filen]
```

Claude guider dig igennem hele opsætningen trin for trin — præcis som en personlig udvikler.

### Trin 4 — Deploy og brug
Følg Claudes instruktioner. Om ca. 30-60 minutter har du din egen personlige tracker live på en URL du kan bogmærke på telefonen.
